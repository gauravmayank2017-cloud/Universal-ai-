# ⚡ GHOST AI - The Universal Action Agent

GHOST AI is a high-performance executive assistant that combines **Google Gemini 1.5** with **Firebase** to not just talk, but **act**.

## 🌟 Why this is the future?
- **Cognitive Memory:** Uses Firebase to remember every interaction.
- **Task Extraction:** Automatically converts natural language into database entries.
- **Ultra Fast:** Powered by Gemini Flash for sub-second responses.

## 🛠 Setup
1. Clone the repo.
2. `npm install`
3. Add your keys in `config.js`.
4. `npm start`
